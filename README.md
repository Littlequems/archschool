# ARCHSCHOOL
É necessário configurar a conexão do banco postgres com o .env para acessar a aplicação.......

#### Na raíz do projeto, criar um arquivo .env
```
DATABASE_URL=postgres://{user}:{password}@{hostname}:{port}/{database-name}

```
